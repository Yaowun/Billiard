# Billiard
## 專案介紹
本專案使用 Unity 遊戲引擎配合 C# 程式語言開發一款花式撞球遊戲。

## 專案技術
Unity 2022.3.0、C# 9.0、Blender 4.0

## 遊戲操作
* 滑鼠左鍵：操控球桿在母球連線方向上的移動以及擊球
* 滑鼠右鍵：操控玩家視角以母球為中心旋轉
* 滑鼠滾輪：縮放玩家視角
